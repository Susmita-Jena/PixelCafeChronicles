# ☕ Pixel Café Chronicles

> *"Every cup tells a story."*

A cozy pixel-art café simulation game where players restore a forgotten café, meet charming townspeople, and create a relaxing place one cup of coffee at a time.

---

## 🌸 Features

- ☕ Brew and serve drinks
- 🌷 Build friendships with NPCs
- 🪴 Decorate your café
- 🌙 Dynamic day & night cycle
- 🌧️ Weather system
- 📱 Mobile & Desktop support
- 💾 Save progress locally

---

## 🛠️ Tech Stack

- Phaser.js
- JavaScript
- HTML5
- CSS3
- Git & GitHub
